# 图片生成方式记录

课程配图必须使用直接生图能力生成 PNG 位图，不使用 Chrome 截图，不使用 SVG 占位图。

## 固定流程

1. 先写文章正文。
2. 从正文提炼信息图结构。
3. 直接调用图像生成能力生成 PNG。
4. 将生成图片保存到 `assets/`。
5. 中文文章只引用中文图片，英文文章只引用英文图片。

## 命名建议

```text
assets/lesson-56-ppt-slide-deck-workflow-zh.png
assets/lesson-56-ppt-slide-deck-workflow-en.png
```

## 提示词原则

- 图片是课程信息图，不是随机封面。
- 使用浅色背景、清晰分区、少量强调色。
- 包含课程标题、关键步骤、使用的 Skill、人工确认点和交付物。
- 不出现 Chrome 标志、浏览器 UI 截图或 SVG 风格占位图。

这条记录用于防止后续生成课程图时误用 Chrome/SVG 路线。
